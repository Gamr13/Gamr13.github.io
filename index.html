- The Data.rsdk file for Sonic CD.
- The video folder for Sonic CD (Optional).

Step 1: Getting the Data.rsdk file.
You can find this through the Android App or more easily, the Steam version of the game.

Step 1: On Android, open up your favourite file manager and open the Android folder

Step 2: obb > com.sega.soniccd.classic > patch.obb & video folder **Note: You need to run the game at least once**

SONIC 1

Read below for instructions.

Pre-requisites:
- The Data.rsdk file for Sonic 1.

Step 1: Getting the Data.rsdk file.
- Download the Sonic 1 app from an Android device.
- Download This app.
- Extract the Sonic 1 app.
- In your files app (I recommend Solid Explorer) go to the ExtractedApks folder > Sonic_1 > Rename the Sonic_1_base.apk to Sonic_1_base.ZIP
- Open the zip file and go to the assets folder
- Press and hold on Data.rsdk.xmf and press the bottom left icon (two boxes) and swipe to the left in Solid Explorer.
- Press the yellow icon on the bottom right of your display to paste it in the INTERNAL MEMORY of your phone.
- Now rename the Data.rsdk.xmf file to Data.rsdk and copy it to your PC or Xbox directly.

Step 2: Where to place the Data.rsdk file.
Use Durango FTP from the store to place the Data.rsdk file in the following folder path:
/LOCALFOLDER/8483Dark-Fox.Sonic-1_e1k75392ey8py/LocalState

Your folder should now look like this (Note: The settings.ini and Udata.bin files are created after running the game): Screenshot Guide by: speedloader57#8037

OPEN LARA

Read below for instructions.

Pre-requisites:
- Assets for Tomb Raider 1 and / or 2.

How to set OpenLara up:
- Connect to your Xbox using Durango FTP and an FTP client of your choice.
- Browse to: LOCALFOLDER/8483Dark-Fox.Open-Lara_e1k75392ey8py/LocalState
- Copy the following folders from Tomb Raider 1 or Tomb Raider 2 into here:
audio, DELDATA, FMV, PSXDATA
- For Tomb Raider 2:
audio, data, FMV, PIX, PIXUS
- Now start OpenLara

Guide by: speedloader57#8037
